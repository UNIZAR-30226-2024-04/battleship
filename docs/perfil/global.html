actualizarEstadisticas(req, res)

Actualiza las estadísticas de un perfil identificado por _id o nombreId. Las nuevas estadísticas indicadas incrementan a las existentes en la base de datos. Los nuevos trofeos se suman si victoria es 1 y se restan si victoria es 0.
Parameters:
Name Type Description
req Object El objeto de solicitud HTTP.
Properties
Name Type Attributes Default Description
body._id string <optional>
 El perfil debe existir en la base de datos.
body.nombreId string <optional>
 El perfil debe existir en la base de datos.
body.victoria number Debe ser 0 o 1.
body.nuevosBarcosHundidos number
body.nuevosBarcosPerdidos number
body.nuevosDisparosAcertados number
body.nuevosDisparosFallados number
body.nuevosTrofeos number <optional>
 0
res Object El objeto de respuesta HTTP.
Source:

Example
perfil = { nombreId: 'usuario1', victoria: 1, nuevosBarcosHundidos: 1, nuevosBarcosPerdidos:1, 
            nuevosDisparosAcertados: 1, nuevosDisparosFallados: 1, nuevosTrofeos: 30};
const req = { body: perfil };

moverBarcoInicial(req, res)

Mueve y/o rota el barco indicado por barcoId en el tablero inicial de un perfil identificado por _id o nombreId. Para trasladar el barco se indica la nueva posición de la proa (parte izquierda si el barco es horizontal o parte superior si es vertical) con iProaNueva, jProaNueva. Para rotar el barco se escribe 1 en rotar.
Parameters:
Name Type Description
req Object El objeto de solicitud HTTP.
Properties
Name Type Attributes Default Description
body._id string <optional>
 El perfil debe existir en la base de datos.
body.nombreId string <optional>
 El perfil debe existir en la base de datos.
body.barcoId number <optional>
 0 Debe estar entre 0 y el número de barcos - 1 (entre 0 y 4 en el juego base).
body.iProaNueva number <optional>
 Debe estar entre 1 y 10.
body.jProaNueva number <optional>
 Debe estar entre 1 y 10.
body.rotar number <optional>
 Debe ser 0 (equivalente a omitirlo) o 1.
res Object El objeto de respuesta HTTP.
Source:
  • perfilController.js, line 328
Example
perfil = { nombreId: 'usuario1', barcoId: 3, iProaNueva: 1, jProaNueva: 6, rotar: 1 };
const req = { body: perfil };
